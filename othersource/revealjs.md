# Reveal.jsでスライドを作る
* Markdownでスライドが書ける**
* スライドをめくる場合は↓と→がある。
* スライドの設定（テーマやスプライト等）を変更できる
* VSCodeならプレビューもできる

# Reveal.jsでスライドを作る
**Markdownでスライドが書ける**
* スライドをめくる場合は↓と→がある。
  * 縦スライドさせる場合は--(-が2つ)
  * 横スライドさせる場合は---(-が3つ)
* スライドの設定（テーマやスプライト等）を変更できる
* VSCodeならプレビューもできる
